# Java QAP3 - Problem Set

This repository contains solutions to the Java QAP3 problem set

## How to Run
1. Clone the repository:
    ```
    git clone https://github.com/Magret1730/abiodun_java_qap3.git
    ```
2. Navigate to the project directory:
    ```
    cd abiodun_java_qap3
    ```
3. Compile and run the Java files:
    ```
    javac ProblemX.java
    java ProblemX
    ```

## Requirements
- Java Development Kit (JDK) 8 or higher
- IDE or text editor of your choice
